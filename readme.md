##  Lending Ledger 
we can create,read and delete data from google sheet

### Dependencies
- dotenv
- express
- googleapis
- zod

### Getting Started
- Clone this git repo
- Run `npm i`
- Add env SHEET_ID , GOOGLE_API
- Add your credentials in credentials.json
- run `nodemon main `
